# Docspyre AI - Web Application

The main web application for Docspyre AI, built with Next.js and Tauri for desktop support.

## Getting Started

Run the development server:

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Desktop Application

To run the Tauri desktop application:

```bash
pnpm tauri:dev
```

To build the desktop application:

```bash
pnpm tauri:build
```

## Features

- AI-powered document processing
- Document tools and management
- Audio/video processing
- Research and academic tools
- Legal and enterprise features
- Authentication and user management

## Tech Stack

- **Framework**: Next.js 16 with App Router
- **UI**: React 19, TailwindCSS, Framer Motion, GSAP
- **Desktop**: Tauri
- **Styling**: Bootstrap 5, Material Symbols
- **Language**: TypeScript
